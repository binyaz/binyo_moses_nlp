# binyo_moses_nlp

Sentiment Classification for Amazon tweets

Determine the sentiment of a tweet using nlp methods

Packages used include
    
packages for EDA

   - numpy
   - pandas
   - regex
   - string

packages for visualizations

   - matplotlib
   - seaborn

packages for preprocessing and modeling

   - sklearn
   - xgboost
   - nltk
   - gensim
